Ma Golide Mothership — The Parent of the Gold Universe

"The one who sees all satellites, consults the Assayer, and builds the Accas"

This is the central brain of the entire Gold Universe. It reads upcoming games from all Satellites, pulls the latest purity report from the Assayer, and builds Accas, Risky Accas, Portfolios, and Bet Slips.

  • Mothership_Genesis.gs — One-click setup for the entire Google Sheet structure
  • Mothership_Menu.gs — Custom Google Sheets menu for manual triggers
  • Mothership_AssayerBridge.gs — Multi-satellite bridge; pulls purity reports
  • Mothership_AccaEngine.gs — Core accumulator building logic
  • Mothership_HiveMind.gs — KPI overview and aggregate statistics
  • Mothership_Intelligence_Core.gs — Advanced intelligence and scoring engine
  • Mothership_RiskyAccaBuilder.gs — High-risk, high-reward accumulator builder
  • Mothership_RiskyAnalyzer.gs — Risk analysis and edge filtering
  • ConfigLedger_Mothership.gs — Config management and purity metadata
  • Performance_Analyzer.gs — ROI and success rate monitoring
  • Systems Audit.gs — System health checks

How to deploy these scripts to Google Sheets

  1. Open (or create) your Ma Golide - MOTHERSHIP Google Sheet.
  2. Go to Extensions → Apps Script.
  3. For each .gs file shown here, create a new script file and paste the contents.
  4. Save the project, then run setupMothership() from Mothership_Genesis.gs once to build all required sheets.
  5. Use the custom Ma Golide menu that appears in your Sheet to trigger syncs, build Accas, and more.
